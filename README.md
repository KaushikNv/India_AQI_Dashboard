# 🇮🇳 India AQI & EV Analysis Dashboard

An interactive data analytics project exploring the relationship between **Air Quality Index (AQI)** and **Electric Vehicle (EV) adoption** across Indian states.

---

## 🚀 Project Overview

This dashboard provides insights into:

- 🌫️ Air quality trends across states
- ⚡ EV adoption patterns
- 🔗 Relationship between EV growth and pollution
- 📊 State-level comparisons
- 🗺️ Geographic AQI visualization

---

## 🔍 Key Insights

- Delhi emerges as the most polluted state based on average AQI
- Several states fall under "Unhealthy" AQI categories
- EV adoption is higher in densely populated states
- No strong direct correlation between EV adoption and AQI reduction yet

---

## 🛠️ Tech Stack

- **Python**
- **Pandas** – Data cleaning & transformation
- **Plotly** – Interactive visualizations
- **Streamlit** – Dashboard development

---

## 📊 Features

- 📌 Interactive state filters
- 📉 EV vs AQI scatter analysis
- 🟥 Top polluted states visualization
- 🟩 Cleanest states comparison
- 🗺️ India AQI choropleth map
- 📈 Year-wise AQI trend analysis
